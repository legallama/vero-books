Team & Security

Secure your organization and collaborate with your team.

User Roles

VeroBooks uses role-based access control (RBAC) to ensure users only see what they need to.

Role Permissions
ADMIN Full access to all settings, financial data, and team management.
ACCOUNTANT Full access to ledger and reports. Cannot manage organization settings or users.
STAFF Can create invoices and record expenses. Cannot view reports or the General Ledger.

Developer & API Access

For advanced integrations, VeroBooks provides a robust REST API. You can generate **API Keys** in the Developer section under your User Profile.

Webhooks

Configure webhooks to receive real-time notifications for events like invoice.paid or customer.created.

Protecting Your Data

Your financial security is our priority. We recommend the following best practices:

  • Rotate API Keys every 90 days.
  • Review the Audit Log weekly to monitor for unusual administrative activity.
  • Limit Admin Roles to only the individuals who absolutely need them.
